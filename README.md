# Arduino Blinking LED

A simple Arduino-based LED project that controls one or more LEDs using basic programming logic.

## Features
- Turns LED on and off using code
- Can be expanded to multiple LEDs
- Handles basic timing control

## Technologies Used
- Arduino (C/C++)
- Resistors
- LED
- Breadboard & Jumper Wires

## What I Practiced
- Digital output control
- Delay Logic
- Basic circuit building
- Debugging hardware connections
- Microcontroller programming

## Future Improvements
- Add button-controlled LED modes
- Create multiple blink patterns (morse code, sequences)
- Upgrade to RGB LED control system
